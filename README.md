Файлы с написанным кодом находятся в папке практическая1. в ней находятся файлы:
index.html- основная страница сайта,
aboutus.html- сводка информации о нас,
contacts.html- форма обратной связи и информация для связи,
korzina.html- корзина в товарами.

В папке css есть файл style.css, в котором прописаны стили сайта.
В папке js лежит файл script.js. тут прописана логика сложных элементов сайта.
В папках icon, images, imagesSales лежат изображения иконки сайта, изображений основных элементов сайта и изображения для карточек товаров соответственно.

Внутри практическая1 есть папка catalog. в ней находяться файлы:
audio.html- страница об аудио,
console.html- страница о консолях и подобных элементах,
fitness.html- страница о фитнесе,
pc.html- страница о пк,
phones.html- страница о смартфонах,
smart_home.html- страница об умном доме,
transport.html- страница о транспортных средствах.

Остальные папки и файлы были созданы VisualStudio автоматически и приложены сюда в избежании потери важного для работы файла.
